## アプリケーション起動

```bash
$ pwd
~/Github/TrainingHackers/webapp

# 省略した起動 ポート 5000
$ carton exec -- plackup app.psgi

# ファイルの変更のたびに再起動
$ carton exec -- plackup -r app.psgi

# 指定しての起動
$ carton exec -- plackup --port 3000 -app app.psgi
```
